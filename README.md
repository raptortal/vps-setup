# vps-setup

Скрипт первоначальной настройки VPS Ubuntu (утилиты + BBR + fail2ban)

## Установка

```bash
wget -O- https://raw.githubusercontent.com/raptortal/vps-setup/main/setup.sh | bash

> ⚠️ **Внимание:** Скрипт предложит сменить пароль для пользователя root (можно пропустить)

---

## 🔧 Что делает скрипт

- ✅ **Подготовка** — устанавливает `wget` и `curl`
- 🔐 **Смена пароля** — предлагает сменить пароль root (опционально)
- 🔄 **Обновление** — `apt update && apt upgrade`
- 🛠️ **Утилиты** — устанавливает набор полезных инструментов
- 🛡️ **fail2ban** — включает защиту от брутфорса SSH
- ⚡ **BBR** — устанавливает TCP BBR + оптимизация сети
- 🔁 **Перезагрузка** — предлагает перезагрузить сервер

---

## 📋 Устанавливаемые утилиты

| Утилита | Назначение |
|:--------|:-----------|
| `htop` | 📊 Мониторинг процессов |
| `nano` | 📝 Текстовый редактор |
| `speedtest-cli` | 🌐 Тест скорости интернета |
| `mtr` | 🔍 Трассировка + пинг |
| `traceroute` | 🛤️ Трассировка маршрута |
| `iftop` | 📈 Мониторинг трафика |
| `nmap` | 🔎 Сканер портов |
| `fail2ban` | 🛡️ Защита от брутфорса |

---

## 💻 Требования

- Ubuntu 20.04 / 22.04 / 24.04
- Права root
